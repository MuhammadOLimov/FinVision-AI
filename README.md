## Biz nima qilamiz?

| Muammo | Bizning yechimimiz |
|-------|------------------|
| Xodimlar qachon kelib-ketayotganini bilmaysiz | Real vaqtda kelish-ketishni kuzatish + ish joyidan chiqib ketganda bildirishnoma |
| Mijozlar soni, yoshi, jinsi va eng ko‘p tashrif vaqtlari noma’lum | To‘liq mijoz analitikasi: yosh, jins, peak hours, qayta tashrif |
| Navbatlarda uzoq kutish, mijozlar norozi | Kutish vaqtini o‘lchash, xodimlarni optimal taqsimlash |
| Xavfsizlik va ruxsatni nazorat qilish qiyin | Yuzni tanish + kirish-chiqish nazorati |
